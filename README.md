# KAL — Creative Studio

This project is a website for **KAL**, a creative studio specializing in branding and graphics for bold businesses. The site features a modern, dark-themed design with lime accents, utilizing the Diodrum Arabic font.

## File Structure
- `index.html`: The main landing page including hero section, logo marquee, portfolio, services accordion, and footer.
- `contact.html`: A dedicated contact page with a "Mad Libs" style form.
- `styles.css`: The central stylesheet containing all layout, typography, and component styling.
- `main copy.js`: The JavaScript file handling interactivity such as mobile navigation, accordions, and form validation.
- `logo.png`: The studio logo asset.
